What Is AniMottos?

AniMottos is a character-development experience designed for children and families who value curiosity, creativity, responsibility, courage, honesty, integrity, and more. Each animal mentor represents a core value and invites children to explore that value through stories, journaling, art, conversation, and real-life practice. Some families begin with the quiz. Others start with the workbook, the stories, or a favorite animal.
